# Contributing to Swap Skills

We love contributors! Here's how to get started:

## Getting Started
1. Fork the repo
2. Clone: `git clone https://github.com/YOUR_FORK/swap-skills.git`
3. Install: `npm install` in both `server/` and `frontend/`
4. Follow setup in README.md

## Finding Issues to Work On
- Start with issues labeled **`good-first-issue`**
- Check **`help-wanted`** for active needs
- Comment on an issue saying "I'd like to work on this"

## Making a PR
- Keep changes focused to one issue
- Write clear commit messages
- Test locally before pushing
- Link your PR to the issue it fixes

## Questions?
Open an issue or reach out! We're welcoming to new contributors.
